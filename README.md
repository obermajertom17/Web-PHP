# Web PHP Profile

Jednoduchý PHP profil, který načítá data z `profile.json` a generuje HTML na straně serveru (bez JavaScriptu).

Jak spustit lokálně (v Codespaces nebo jiném prostředí):

1. Otevřete terminál v adresáři projektu.
2. Spusťte vestavěný PHP server:

```bash
php -S 0.0.0.0:8080 -t .
```

3. Otevřete v prohlížeči: `http://localhost:8080` nebo v Codespaces použijte forwarded port.
